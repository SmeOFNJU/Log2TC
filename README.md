# Log2TC
a research project on how to make use of logs to generate, optimize test cases
## Main Content
* An automatic generation framework of test case
* Test case optimization based on ontology
* Research on test case prioritization
## Dependency
* matlab && python && Java
* SqlServer
* Protege
* Jena
* GCC
* ...
